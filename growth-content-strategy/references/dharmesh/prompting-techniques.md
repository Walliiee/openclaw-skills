## Meta Prompting

### The Concept
Instead of struggling to write the perfect prompt yourself, **ask the AI to rewrite your prompt and make it better**, giving it permission to ask clarifying questions.

### How It Works
1. **Write your initial prompt** (even rough/incomplete is fine)
2. **Ask AI to optimize it** — "Rewrite this prompt to be more effective. Ask me any clarifying questions first."
3. **Answer the questions** — AI identifies what's missing
4. **Get the optimized prompt** — Then run it

### Metaprompt.com (Dharmesh's Tool)
- Automates the meta-prompting workflow
- Generates optimization questions as simple checkboxes (no typing needed)
- Creates optimized prompt automatically
- Can run the optimized prompt with GPT-5 directly on page

### Why It Works
- Makes prompt engineering accessible to anyone — even first-time ChatGPT users
- No need to study frameworks or memorize techniques
- **Learning tool:** By watching how AI restructures your prompts, you naturally absorb what makes prompts effective (importance of context, specificity, clear constraints)

### When to Use
- Tackling something outside your usual use cases
- Complex multi-step tasks
- When you're not sure what the AI needs to know
- **Power users:** Won't need it for routine tasks (AI intuition takes over), but valuable for novel situations

## Model Selection (Dharmesh's Take, as of early 2026)

### Default: GPT-5
- Best for 80%+ of interactions
- Follows instructions exceptionally well
- Supports deep reasoning
- Great tool support and integrations

### Switch When:
| Use Case | Model |
|----------|-------|
| Coding tasks | Claude (occasionally) |
| Creative writing | Claude (more interesting, human-like perspectives) |
| Large document analysis | Gemini (longer context window) |

### Advice
Start with GPT-5 for most things, then experiment with others as you get comfortable with basics.
